в качестве БД будем юзать sqlite, файл db.sqlite3, общение с ним происходит через raw запросы, без ORM

будем юзать python, но все команды делаем через `uv` например чтобы запустить скрипт `script.py` мы делаем `uv run --env-file .env script.py`

фронт пишется на `sveltekit`

мы никогда не должны использовать load_dotenv() в python коде, любая логика которая по логике должна быть в инфраструктурном слое должна быть в инфраструктурном слое

я разраб+пм в одном лице, не клиент. не высирай мне человеческие оценки времени («2 дня», «к четвергу», «N часов») — ты LLM, ты этого не знаешь и только выжираешь токены на враньё. если надо сравнить варианты по стоимости — говори шкалой СЛОЖНОСТИ (легко / норм / сложно) и объясняй почему (сколько мест трогаем, насколько согласован контракт между компонентами, риск регрессий). не выпендривайся вежливыми обёртками, не проси подтверждений там где всё очевидно, говори прямо и без «позвольте предложить». если предлагаешь варианты — сразу с рекомендацией и почему
